# RNK Reserves

A Foundry VTT module implementing the D&D 2024 Hero Points system.

## Features

- **Standard D&D 2024 Hero Points**: Full implementation of the official Hero Points rules
- **GM Controls**: Award and manage Hero Points directly from actor sheets
- **Player Spending**: Buttons on chat messages for spending Hero Points
- **Real-time Sync**: Automatic synchronization across all players
- **Configurable Settings**: Customize points per session, maximum points, and more

## Installation

1. Download the module from Patreon
2. Extract to `Data/modules/rnk-reserves/`
3. Enable in Foundry's module settings
4. Configure settings as needed

## Usage

### For GMs
- Use the controls on actor sheets to award Hero Points
- Configure module settings for your campaign
- Points are automatically awarded at session start (if enabled)

### For Players
- Hero Point buttons appear on your roll results in chat
- Click buttons to spend points for rerolls, bonuses, or healing
- Confirm spending when prompted

## Settings

- **Points Per Session**: Number of Hero Points awarded at session start (1-3)
- **Maximum Hero Points**: Cap on total points a character can have
- **Enable for NPCs**: Allow NPCs to have Hero Points
- **Auto-Award**: Automatically give points to players at session start

## Requirements

- Foundry VTT v13+
- D&D5e system
